  • Introduction

    Introduction

    This course will focus on building web applications using the Python / Django Web Application Framework. In addition to Python and Django, the course will involve a range of other technologies including HTML, CSS, XML, JavaScript, jQuery and AJAX.

    • Course textbook

      Course textbook

      This course will be based on the book, "How to Tango With Django" by Leif Azzopardi and David Maxwell.  The book is hosted online at: http://www.tangowithdjango.com (make sure you choose "Book 1.9/1/10").  It can be accessed free of charge by University of Glasgow staff and students here.  The access code is yMNpPQXt.  You must not share this access code with anyone outside of the University of Glasgow.
